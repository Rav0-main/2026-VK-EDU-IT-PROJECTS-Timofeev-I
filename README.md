# Домашнее задание №1

## Идея

**"Личный" менеджер по отслеживанию питания**.

## Elevator Pitch

Большинство людей в мире страдает ожиранием и некачественным питанием.\
Одно из возможных решений - это составление и оценка питания.\
В отличие от других приложений, мы учитываем всевозможную информацию о пользователе: его локацию и потребление продуктов. В результате клиент получает исчерпывающие рекомендации.

## Lean Canvas

### 1. Сегменты потребителей

* Люди, имеющие проблемы со ожирением или желающие следить за своим питанием, к примеру: профессиональные спортсмены.
* Специалисты, занимающиеся рекомендациями питания.
* Сети питания и магазины для рекламы их в приложение.
* Рекламодатели, к примеру спортивные залы.

### 2. Проблема

* Неудобство системно отслеживать питание.
* Проверка соблюдения рекомендации специалистов по питанию.
* Поиск заведений с меню правильного питания.

### 3. Уникальная ценность

**Предложение**:
> Проверь, что и где можно поесть, не вредив своему здоровью.

**Конкуренты**:
* Самозанятые эксперты по питанию.
* Приложения по статистике о здоровье: Samsung health, Apple health и т.д.

### 4. Решение

* Добавление нейронных сетей для быстрых рекомендаций. Условно, сфоткал продукт (котлета с картофильным пюре) - получил калорийность и рекомендации о потребление.
* Разрешаемый просмотр о покупаемых продуктов для информации о потребляемой еды и предупреждениях, если товар не соответсвует плану.
* Поиск нужных магазинов и заведений питания.
* Возможность иметь быструю связь с экспертами по питанию.

### 5. Каналы

* Информационные поля, специализирующиеся на здоровье: телеграм-каналы, сайты, видеокреаторы.
* Составление рекламы на основе предоставленной пользователем информации.
* Рестораны, столовые и т.д.

### 6. Поток прибыли

* Улучшение качества за счёт подписки:
    * Улучшение рекомендаций.
    * Возможность связаться к специалистам по питанию.

* Плата от рекламодателей: продвижение магазинов или заведений правильного питания.

### 7. Структура издержек

* Эксплуатация и поддержка приложения.
* Часть платы от подписок пользователя передается экспертам по питанию.
* Реклама приложения для пользователей и экспертов по питанию.

### 8. Ключевые метрики

* Количество экспертов на количество пользователей.
* Частота полезных советов, приходящих к пользователю.
* Количество положительно/отрицательно оцененных рекомендаций: экспертов по питанию и сетей питания.
* Количество и время заходов в приложение в день.

### 9. Скрытое преимущество

* Полнота: не только количество шагов в день.
* База зафиксированных (по своему желанию) экспертов и пользователей.
* Чем больше пользователь использует приложение, тем лучше рекомандации для него.
* Чем больше качественных рекомендаций, тем больше устоявшихся пользователей: нежелание переходить на аналоги.

# Домашнее задание №2

## Roles & User Stories

### Roles

* Пользователи, желающие улучшить свой рацион питания.
* Эксперты по составлению питания.
* Заведения и магазины правильного питания.
* Поддержка.

### User Stories

#### Пользователи, желающие улучшить свой рацион питания.

1) Создать/удалить профиль пользователя с личной информацией и целями.
2) Создать план потребления продуктов. Условно:
    * Понедельник:
        * Завтрак: 350 ккал, 30/0/5 г БЖУ
        * Обед: 600 кклал, 70/20/10 г БЖУ\
        ...
3) Записать, что и в каком количестве было съедено.
4) Просмотреть аналитику своего питания, в точности соблюдения плана от специалиста или составленного лично.
5) Оценить калорийность продукта.
6) Просмотреть [ ближайшие ] сети питания с соответсвующими рекомендациями по меню.
7) Купить/отменить подписку.
8) Проконсультироваться/закрепиться со специалистами по питанию.
9) Поставить оценку специалисту за рекомендации.

#### Эксперты по составлению питания.

1) Зарегистрировать/забыть себя, как специалиста по питанию.
2) Получить контакты клиента для оказания услуг с ним.
3) Сделать план по питанию.
4) Проверить качество соблюдения плана определенного клиента.

#### Магазины и заведения правильного питания.

1) Зарегистрировать/удалить себя и свой ассортимент.
2) Получить рекламу себя в приложении.

#### Поддержка

1) Разрешить технические проблемы пользователей.
